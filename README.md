# Happy Birthday Princess Oluwabukolami

A romantic birthday website built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit Content

- Main messages, wishes, reasons, gallery captions, and music settings are in `src/data/siteData.js`.
- Put personal photos in `public/assets`, then update each gallery `src` value, for example `/assets/photo-1.jpg`.
- The hero photo is configured as `public/assets/image.png`.
- Add the song file as `public/love-song.mp4`, or change `backgroundSong` in `src/data/siteData.js`.

## Generated Asset

The hero background is saved at `public/assets/romantic-hero.png`.
